
# Coxe v2.0 Beta — Code Operations eXchange Environment

**Coxe** 是一款完全运行在浏览器中的纯本地代码协作与重构沙盒。它能够：

- 载入本地的代码文件或 ZIP / tar.gz 压缩包，形成一个可编辑的虚拟工作区。
- 通过 JSON 过滤规则（扩展名、路径）智能忽略大文件或第三方依赖，保留拓扑结构，节省 AI Token 消耗。
- 提供内置的 **PITCH 修补引擎**，支持 AI 通过 XML 指令对工作区进行创建、修补、删除、重命名、移动、复制操作。
- 拥有 **时光机** 历史快照系统，可随时回滚到任意历史版本；支持 **非线性版本拓扑图**，可拖拽、缩放、点击回滚。
- 内置 HTML 预览与编译下载（将虚机内的 JS 依赖内联到单文件）。

## 🚀 V2.0 Beta 新增功能（相对于 V1.0）

### 1️⃣ 项目管理器
- 支持创建多个独立项目，每个项目拥有专属的文件快照、时光机历史、过滤配置和 AI 模型配置。
- 提供顶部导航栏的 **项目选择器**，可快速切换项目。
- 支持**导出/导入**整个项目（包含全部文件、历史快照、对象数据库和配置），便于备份和分享。
- 允许重命名或删除项目（删除前二次确认）。

### 2️⃣ 内置 AI 智能对话助手
- 点击右下角聊天按钮打开 **AI Chat 抽屉**，可直接与 AI 模型对话。
- 支持配置多个 AI 模型（如 DeepSeekChat、DeepSeekCoder），每个模型可单独设置 API 地址、Key、System Prompt 和记忆模式（有状态 / 无状态）。
- 支持 **流式响应**（实时展示正在生成的文字），并在对话气泡中自动解析 XML `<file>` 指令块，生成美观的可交互卡片（一键“运行修改”）。
- 对话中可通过 `@模型名` 快速切换当前活跃模型，无需离开输入框。
- **全局 API Key** 输入框，所有模型统一使用（或每个模型可覆盖独立 Key）。

### 3️⃣ 增强的时光机与日志系统
- 每次执行 XML 动作时，自动创建 **日志分组**（Log Group），记录该次操作的所有控制台输出、操作详情、执行前后的文件快照。
- 在 AI 对话中，可通过 `{{日志组:...}}`、`{{失败日志:...}}`、`{{日志组代码对比:...}}` 等变量引用这些日志数据，让 AI 感知执行历史。
- 时光机增加“编辑备注”功能，可随时修改快照的备注名称。
- 版本拓扑图支持更丰富的交互：鼠标拖拽平移、滚轮缩放、缩放指示器、自适应居中复位。

### 4️⃣ 智能变量注入与级联建议
- 在 AI 输入框中输入 `{{` 会弹出**级联变量面板**，逐级选择当前工作区、单文件、历史快照节点或日志组，插入 XML 格式的完整数据。
- 示例：`{{当前全部文件}}` 插入整个工作区的 XML 大纲；`{{节点: nodeId:全部文件}}` 还原快照时刻的全部文件。
- 变量插入后，在对话历史中渲染为美观的彩色芯片（Chip），点击可查看详情。

### 5️⃣ 其他改进
- 配置文件现在支持 `aiAgents` 数组，同时保留了 `hidePaths`（AI 视图屏蔽目录）功能。
- 文件删除改为**两段式确认**：第一次点击变红，第二次确认删除，防止误操作。
- 复制单文件时使用自动计算的选择范围，适配超长文件。
- 优化了整体 UI 布局（顶部导航栏增加项目栏位、统计信息更清晰）。
- 时光机对象数据库（CAS）支持多项目独立存储，自动清理孤立内容。

## 🧪 快速开始

1. 双击 `index.html` 直接在浏览器打开（无需任何服务器）。
2. 默认载入演示项目，包含一个 `node_modules`、`mp3` 等被过滤的大文件演示。
3. 尝试切换选项卡：**目录树查看器**、**AI 字符串源数据**、**XML 动作指令执行器**。
4. 点击右下角聊天按钮，输入 API Key 后即可与 AI 对话，并使用 `@` 切换模型。

## 📦 技术栈

- 纯浏览器端运行，无后端依赖。
- 使用 Tailwind CSS (CDN) 做样式。
- 使用 JSZip 处理 ZIP 解压/打包。
- 使用 pako 处理 gz / tar.gz 解压。
- 使用 Fetch API 与 AI 模型交互（OpenAI 兼容接口）。
- 所有数据通过 LocalStorage 持久化（可禁用）。

## ⚙️ 配置说明

在 “AI 字符串源数据” 选项卡中，您可以编辑 JSON 配置：

```json
{
  "excludeExtensions": [".mp3", ".png", ...],
  "excludePaths": ["node_modules/", "dist/", ...],
  "maxHistoryLimit": 10,
  "hidePaths": [],
  "enableLocalStorage": true,
  "aiAgents": [
    {
      "id": "agent_deepseek_chat",
      "name": "DeepSeekChat",
      "systemPrompt": "...",
      "apiUrl": "https://api.deepseek.com/v1/chat/completions",
      "apiKey": "",
      "model": "deepseek-chat",
      "type": "stateless"
    }
  ]
}
```

保存后立即生效，并持久化到当前项目。
